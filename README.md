# Kimia Farma Performance Analytics (2020–2023)

Project ini bertujuan untuk menganalisis kinerja bisnis Kimia Farma
periode 2020–2023 berdasarkan data transaksi, produk, dan cabang.

Analisis dilakukan menggunakan Google BigQuery untuk membangun
tabel analisa yang berisi informasi penjualan, profit, dan rating,
yang selanjutnya digunakan sebagai sumber data dashboard di
Google Looker Studio.

## Objective
- Mengetahui tren pendapatan Kimia Farma dari tahun ke tahun
- Mengidentifikasi cabang dan provinsi dengan performa terbaik
- Menganalisis hubungan rating cabang dan rating transaksi

## Tools
- Google BigQuery
- Google Looker Studio
